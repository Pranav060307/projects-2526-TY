# Purohit Mitra 🙏 

Purohit Mitra is a digital platform designed to connect users with verified purohits (priests) for performing Hindu rituals, pujas, and spiritual services. The main aim of the project is to simplify the process of finding and booking knowledgeable priests for various occasions such as weddings, housewarming (griha pravesh), festivals, and other religious ceremonies.

The platform allows users to explore different types of pujas, view details, and make bookings easily from their device. On the other side, purohits can register themselves, showcase their expertise, and receive booking requests. The system includes features like user registration, service listings, booking management, and a dashboard to handle all activities efficiently.

Overall, Purohit Mitra acts as a bridge between tradition and technology, making spiritual services more accessible, organized, and convenient for modern users.

## Features
- User booking system
- Pandit dashboard
- Booking management (confirm/reject)
- Email notifications

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## Software Installation Instructions  
1. Objective 
To install and run the React + MongoDB web application successfully on a system. 
2. Requirements 
• Computer / Laptop 
• Internet Connection 
• Node.js installed 
• MongoDB Compass installed 
• Code editor (VS Code recommended) 
• Project ZIP file 
3. Procedure / Steps 
. Unzip the project folder to a desired location on your computer. 
. Install Node.js from the official website if not already installed. 
. Go to the frontend folder and open terminal. 
. Run the command 'npm install' to install all required dependencies. 
. Run 'npm start' to start the frontend. 
. Ensure MongoDB is installed and running OR connect using MongoDB Compass. 
. Configure database connection if required (check .env file). 
. Go to backend folder and run 'npm install' to install server dependencies. 
. Run the backend server using command 'node app.js' or 'nodemon app.js'. 
. Open browser and access the application at http://localhost:3000. 
4. Expected Output 
The React application will run successfully in the browser and connect to MongoDB database. 
5. Conclusion 
The React + MongoDB project has been successfully installed and executed. 
6. Precautions - Always unzip the folder before running. - Install Node.js properly before executing commands. - Ensure internet connection is active. - Do not skip npm install step. - Make sure MongoDB Compass is connected properly. - Check for errors in terminal and resolve them. - Do not change project files unnecessarily.
